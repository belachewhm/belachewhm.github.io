# Billy Haile-Mariam — Personal Portfolio

Personal portfolio and demo site for Belachew "Billy" Haile-Mariam, Lead Software Engineer.

## Stack

- **HTML/CSS/JS** — no build step, no framework
- **Bootstrap 5** — layout and components (CDN)
- **Font Awesome 6** — icons (CDN, with v4 shim)
- **Google Fonts** — Merriweather + Open Sans (CDN)

## Pages

- `index.html` — main portfolio (about, skills, experience, education, resume, contact)
- `login.html` — demo login page (always rejects auth)

## Running locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.
